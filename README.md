## simple-grpc-server

A simple gRPC server with Node

### Deploy

```shell script
git clone https://github.com/peterdee/simple-grpc-server
cd ./simple-grpc-server
nvm use 14
npm i
```

### Environment variables

The `.env` file is required. See the [.env.example](.env.example) file for details.

### Launch

```shell script
npm start
```
